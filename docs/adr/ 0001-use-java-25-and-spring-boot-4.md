# ADR-0001: Use Java 25 and Spring Boot 4

## Status

Accepted

## Context

InsightFlow is designed as a modern backend architecture project. The project aims to demonstrate current Java and Spring Boot practices while remaining compatible with production-oriented enterprise development.

## Decision

We will use Java 25 and Spring Boot 4 for the backend application.

## Consequences

- The project can use modern Java language and runtime features.
- Spring Boot 4 provides a current foundation for web, persistence, validation, security, observability and testing.
- Developers need a Java 25 compatible JDK installed locally.
